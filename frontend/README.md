# pawsome Frontend
Frontend Application for Pawsome built using React + Vite

## Project Setup
### Tech Stack
- React
- Vite
- React Router DOM


### Install Dependencies

- npm create vite@latest after this choose
- FrameWork : React 
- Variant : JavaScript 
- npm install 

#### To Start the Server
- npm run dev

- Upon successfull creation our project run on : http://localhost:5173/
- after this install React Router to install this React Router run the command : npm install react-router-dom
- our Project Structure is ready
